Dieses Notebook lädt ein Beispiel-Git-Log (oder eine hochgeladene Datei) 
und bietet interaktive Filter sowie Visualisierungen zu Commits pro Autor 
und pro Tag. 
Anleitung: 
- Lade optional eine `git log`-Datei hoch (Format: `git log --date=iso --pretty=format:%H%x09%an%x09%ad%x09%s`)
- Oder nutze die Demo-Daten - Filtere nach Autor, Datumsbereich und Suchbegriff im Commit-Text
